## String Class

This app hass different methods of modifying strings.

*Install App*

Clone the repository
> git clone https://github.com/wanjiku-carol/string-class.git

cd into directory
> cd string-class

Install app dependencies
> npm install

Run the tests
> jasmine tests/tests_stringclass.js
